# はかる・くらべる・ほりさげる

## はかるもの

### 1. UX（User Experience）

インタラクティブな製品のUX評価（品質）は主に

- **実用的品質**（pragmatic quality）
- **感性的品質**（hedonic quality, affective quality）

に分けられ，アンケート調査によるUX評価が行われている．

### 2. 印象

対象から受ける印象を評価する際は，複数の形容詞やオノマトペを通じて評価する．

### 3. 好み

直感的に良い，しっくりくる，ふさわしいと感じること．

### 4. 感情

感情はP(V)ADモデル（Pleasure (Valence), Arousal, Dominance）がよく知られる．中でも，コンテンツから受け取る感情は，

- **覚醒度**（鎮静 ー 興奮）
- **快不快**（ポジティブ ー ネガティブ）

の2軸で評価することが多い．

## はかる方法

### 1. SD（Semantic Differential）法

作品や製品等から受ける人の印象を客観的に評価・分析する手法．「明るい - 暗い」，「人工的な - 自然な」など，対立する形容詞の対を用いて5段階または7段階の尺度で回答させる（[参考文献](https://www.jstage.jst.go.jp/article/jje/45/5/45_5_263/_pdf/-char/ja)）．

参考: [ARインスタレーションの印象評価](https://www.jstage.jst.go.jp/article/jjske/24/2/24_TJSKE-D-24-00046/_pdf/-char/ja)，[バーチャル森林浴コンテンツの印象評価](https://www.mdpi.com/1660-4601/19/3/1202)，[SD法による調査結果のグラフ化](https://note.com/shinoan_01/n/nd38277730941)

### 2. 半構造化インタビュー

あらかじめ質問項目を定めておき，回答内容に応じてその心理を掘り下げていく調査手法（[参考サイト](https://www.asmarq.co.jp/column/column-cat/glossary/semi-structured_interview/)）．

### 3. フォーカスグループインタビュー（FGI）

1人のモデレーターが複数の調査対象者に対して座談会形式でインタビューを行う調査手法．グループ内の参加者それぞれがお互いの発言に刺激されるため，意見が活発になりやすく多くの意見を収集しやすい（[参考サイト](https://surveroid.jp/mr-journal/interview/kNQI6)）．

### 4. 具体的な測定方法

- [**AttrakDiff**](https://www.attrakdiff.de/index-en.html)
- [**UEQ**](https://www.ueq-online.org/)（またはUEQ-S．[アンケートサンプル](https://forms.gle/4FhLjZGfVuAUm4u56)）
- **SAM**（感情．[アンケートサンプル](https://docs.google.com/forms/d/e/1FAIpQLScWdaTtI_owkrPHgYtuEQ3K04oPTCGoOkwQx-ISzsYszVpsHg/viewform)）
- **POMS2**（ネガティブな気分）
- **Temporal Dominance of Emotion**（感情の経時変化）
- [**PRS**](https://www.researchgate.net/profile/Seiji-Shibata-2/publication/235659586_Perceived_restorativeness_in_everyday_environments/links/0fcfd5125b523bcbb7000000/Perceived-restorativeness-in-everyday-environments.pdf)（場所の特徴・魅力）
- [**PANAS**](https://www.jstage.jst.go.jp/article/jjpjspp/9/2/9_KJ00001287160/_pdf)（快と不快を独立とみなした気分評定尺度）
- **ME（マグニチュード推定）法**

## くらべる・ほりさげる方法

- 作品内の要素／表現手法を変える（比較のためのダミー作品を用意）
- 媒体を変える（モニター ⇔ プロジェクタ，ヘッドホン ⇔ スピーカ）
- 既存の似た作品，製品と比較する
- **コンジョイント分析**（構成要素の好ましさ）
- **テキストマイニング**（アンケート内の語句の頻度や共起性を分析）
- **分散分析**（3種類以上の変数の平均値に差があるか比較）
- **因子分析**（SD法によって評価された印象のカテゴリ分け）
