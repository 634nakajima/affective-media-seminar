# 身の回りの評価・比較

## 身の回りの物事に着目した印象評価・比較

### 目的

身の回りの物事が持つ感性価値を，SD法などを用いて評価

### 手順

1. 興味がある調査対象（物事，人，環境，場所等）を一つ決めてください．その際，なぜその調査対象を選んだのか，理由も考えておいてください．
   - 例：香水のビンの形状，アニメキャラの髪型，自動車の外観...
2. 調査対象に関連しそうな感性情報（印象，形容詞，オノマトペ等）を一つ以上決めてください．
   - 例：可愛らしさ，高級感，シンプルさ，強さ，おとなしさ...
3. 計測・比較をするための具体的な調査対象を示すもの（実物，画像，動画等）を複数用意してください．
4. 感性情報を数値化するアンケートをGoogleフォームで作成してください．
5. ゼミのメンバー同士でアンケートに回答しあって回答を収集してください．
6. 集計結果をグラフにまとめてください．
7. グラフから読み取れることを考察してください．
8. 調査対象，着目した経緯，調査する感性情報，調査結果，考察をスライド資料にまとめてください．

### スケジュール

| Step | 時期 |
|------|------|
| Step 1-2 | 1週目（提出フォーム(1)） |
| Step 3-4 | 2-3週目（提出フォーム(2)） |
| Step 5 | 4週目（ゼミ時間内で実施） |
| Step 6-8 | 5週目（提出フォーム(3)） |

## 身の回りの物事から受ける感情の評価・比較（SAM）

### 目的

身の回りの物事から受ける感情を，SAM（Self-Assessment Manikin）を用いて快不快・覚醒度の2軸で評価・比較

### 手順

1. 感情を喚起しそうな調査対象のカテゴリ（音楽，映像，画像，場所，食べ物等）を一つ決めてください．
2. そのカテゴリの中から，異なる感情を喚起しそうな具体的な刺激を5つ以上用意してください．
3. SAMのアンケートを参考に，各刺激に対する「快不快（Valence）」と「覚醒度（Arousal）」を9段階で回答するアンケートを作成してください．
4. ゼミのメンバー同士でアンケートに回答しあってください．
5. 各刺激の快不快・覚醒度の平均値を算出し，2次元の感情空間上に散布図を作成してください．
6. 散布図から読み取れることを考察してください．
7. スライド資料にまとめてください．

::: info SAMとは
SAM（Self-Assessment Manikin）は，言葉を使わずにイラスト（マニキン）で感情を評価する手法です．主に **快不快（Valence）** と **覚醒度（Arousal）** の2軸で評価します．言葉による表現が難しい感情の微妙な違いを，直感的に数値化できるのが特徴です．
:::

## 空間の魅力の評価・比較（PRS）

### 目的

身の回りの空間・場所が持つ回復感（restorativeness）を，PRS（Perceived Restorativeness Scale）を用いて評価・比較

### 手順

1. 評価したい空間・場所のカテゴリを一つ決めてください．
2. 雰囲気が異なりそうな具体的な空間を4つ以上選び，写真や動画を準備してください．
3. PRSの4因子に関する質問項目を用いたアンケートを作成してください（7段階）：
   - **Being Away**（日常からの離脱感）
   - **Fascination**（魅了）
   - **Coherence**（まとまり）
   - **Compatibility**（適合性）
4. ゼミのメンバー同士で回答しあってください．
5. 各空間の4因子の平均値を算出し，レーダーチャートで可視化してください．
6. グラフから読み取れることを考察してください．
7. スライド資料にまとめてください．

::: info PRSとは
PRS（Perceived Restorativeness Scale）は，ある環境・空間がどれくらい心理的な回復（リフレッシュ）をもたらすかを評価する尺度です．環境心理学のART（Attention Restoration Theory）に基づいています．
:::

## 身の回りの物事の好みの評価・比較（コンジョイント分析）

### 目的

身の回りの物事の好みに影響する要素を，簡易的なコンジョイント分析を用いて明らかにする

### 手順

1. 好みを分析したい対象のカテゴリを一つ決めてください．
2. その対象の好みに影響しそうな属性を2\~3つ挙げ，それぞれの属性に対して水準を2\~3つ設定してください．
   - 例（スマホケース）：属性A「色」→ 黒，白，パステルピンク ／ 属性B「素材感」→ マット，光沢，透明
3. 属性と水準の組み合わせ（プロファイル）を9通り程度に絞って作成してください．
4. 各プロファイルに対する好みを5段階で評価するアンケートを作成してください．
5. ゼミのメンバー同士で回答しあってください．
6. 各属性の各水準ごとに好み評価の平均値を算出し，棒グラフ等で可視化してください．
7. グラフから読み取れることを考察してください．
8. スライド資料にまとめてください．

::: info コンジョイント分析とは
コンジョイント分析は，商品やサービスを構成する複数の要素（属性）のうち，どの要素がどれくらい好みに影響しているかを明らかにする手法です．
:::

## 身の回りの商品・サービスのUX評価・比較

### 目的

身の回りの商品・サービスのUXを，UEQ-Sを用いて実用的品質・感性的品質の観点から評価・比較

### 手順

1. UX評価したい商品・サービスのカテゴリを一つ決めてください．
2. 比較対象となる具体的な商品・サービスを3つ程度選んでください．
   - 例（メモアプリ）：Apple純正メモ，Notion，Google Keep
3. 各商品・サービスを実際に使用してもらうための簡単なタスクを用意してください．
4. UEQ-Sの設問を含むアンケートを作成してください．
5. ゼミのメンバーにタスクを実施してもらった後，アンケートに回答してもらってください．
6. 回答結果を分析用シートにコピペし，実用的品質と感性的品質のスコアを算出・可視化してください．
7. グラフから読み取れることを考察してください．
8. スライド資料にまとめてください．

::: info UEQ-Sとは
UEQ-S（User Experience Questionnaire - Short version）は，8つの形容詞対を用いて製品・サービスのUXを簡便に評価する尺度です．**実用的品質**（使いやすさ，効率性）と**感性的品質**（面白さ，魅力）に分類されます．
:::
