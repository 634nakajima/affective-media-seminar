# コンペ・動画・ポスター

## 主なコンペ・展覧会

作品を外部に発表する機会として，以下のようなコンペティションや展覧会，学会があります．アート寄りのものから学術寄りのものまで幅広いので，自分の作品の性格に合った場を選びましょう．

### アート系コンペ・展覧会

| 名称 | 概要 |
|------|------|
| **SICF**（Spiral Independent Creators Festival） | 表参道スパイラルで開催される若手クリエイターの公募展．アート，デザイン，ファッション，映像など幅広いジャンルが対象．審査を通過するとスパイラルで展示できる |
| **ADAA**（Asia Digital Art Award） | アジア発のデジタルアートの国際公募展．インタラクティブアート，メディアアート，映像，アニメーション等が対象 |
| **TOKAS**（Tokyo Arts and Space） | 東京都が運営するアートスペース．公募展やレジデンスプログラムを通じて若手アーティストの活動を支援．本郷と天王洲に施設がある |
| **やまなしメディア芸術アワード** | 山梨県が主催するメディア芸術の公募展．メディアアート，映像，ゲーム，マンガ，アニメなどが対象 |

### 学術系学会・研究会

| 名称 | 概要 |
|------|------|
| **インタラクション**（情報処理学会） | 情報処理学会が主催するインタラクション研究の国内最大級のシンポジウム．インタラクティブ発表（デモ展示＋ポスター）形式で，実際に作品を体験してもらいながら議論できる |
| **EC**（エンタテインメントコンピューティング） | 情報処理学会のエンタテインメントコンピューティング研究会が主催するシンポジウム．ゲーム，VR，インタラクティブアートなどエンタメ寄りのHCI研究が集まる |
| **バーチャルリアリティ学会大会** | 日本バーチャルリアリティ学会の年次大会．VR/AR，触覚，クロスモーダルなど，身体感覚に関わる研究発表が多い．デモ展示の機会もある |
