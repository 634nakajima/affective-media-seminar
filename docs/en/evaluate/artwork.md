# Artwork Evaluation

## Impression Evaluation of Artwork Experiences

### Purpose

Objectively evaluate the impressions received from an artwork using the SD method

### Procedure

1. Create question items in Google Forms
2. Referring to the [impression evaluation of AR installations](https://www.jstage.jst.go.jp/article/jjske/24/2/24_TJSKE-D-24-00046/_pdf/-char/ja), prepare the following adjective pairs + any additional adjective pairs related to impressions you are interested in:

| Adjective Pair |
|----------------|
| Bright ~ Dark |
| Harmonious ~ Disjointed |
| Cramped ~ Spacious |
| Unique ~ Common |
| Flashy ~ Subdued |
| Lively ~ Lonely |
| Beautiful ~ Ugly |
| Dynamic ~ Static |
| Simple ~ Complex |
| Realistic ~ Fantastic |
| Comfortable ~ Unsettling |
| Passive ~ Active |
| Natural ~ Artificial |
| Vivid ~ Dull |
| Energetic ~ Calm |
| Boring ~ Attractive |
| New ~ Old |
| Memorable ~ Forgettable |
| Emotional ~ Detached |
| Powerful ~ Lacking |

3. Calculate the mean value for each adjective pair from the responses
4. Create a line graph in Excel, referring to [Graphing SD method results](https://note.com/shinoan_01/n/nd38277730941)
5. Using the graph, analyze and discuss whether the artwork conveyed your intent as intended
6. Compile the graph and your analysis into a brief slide presentation (about 1-2 slides)

::: tip When significant impression changes are expected during the experience
If major impression shifts are anticipated, have participants respond at specific points during the experience (when first seeing the artwork, when touching the device, after completing the experience...) to capture impression changes over time
:::

## UX Evaluation of Artwork Experiences

### Purpose

Evaluate the artwork's UX from the perspectives of pragmatic and hedonic quality using UEQ-S

### Procedure

1. Create the same questions as in the [survey sample](https://forms.gle/4FhLjZGfVuAUm4u56) and ask participants to respond after experiencing the work
2. Copy and paste the response results into the [analysis sheet](https://docs.google.com/spreadsheets/d/1JlZK5hIA1e4jaUxD_UCu_NoXzP5Y8ImV/edit?usp=drive_link&ouid=100272312101849018961&rtpof=true&sd=true) (make a copy before use)
3. Using the graphs in the analysis sheet, analyze and discuss the artwork's UX
4. Compile the graphs and your analysis into a brief slide presentation (about 1-2 slides)
